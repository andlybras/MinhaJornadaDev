# Tipos de Condições 
  
Há muitas diversas maneiras de filtrar dados não desejados.  
  
## Condições de Igualdade  

Essa tipologia de condição segue a fórmula `COLUNA = expressão`.   
  
## Condições de Desigualdade 
  
Essa tipologia define que duas expressões não são equivalentes, seguindo a fórmula `expressão <> expressão` ou `expressão != expressão`.  
  
## Condições de Intervalo  
  
São condições que verificam se uma expressão está dentro de um intervalo. A fórmula `expressão < expressão` é um exemplo dessa condiçõe que usa o operador <.  
  
O operador `BETWEEN` é uma opção válida para construir condições de intervalo quando se pretende resultados com limites mínimos e máximos concomitantes. A fórmula dessa expressão segue `expressão BETWEEN expressão_mínima AND expressão_máxima`.
