# Capítulo 3  
  
## Claúsulas de Consulta  
  
> `SELECT` - Determina quais colunas incluir na tabela não persistente;
> `FROM` - Identifica as tabelas  das quais recuperar os dados e como as tabelas podem ser juntadas;
> `WHERE`- Filtra dados indesejados;
> `GROUP BY`- Usado para agrupar linhas de acordo com valores de coluna em comum;
> `HAVING`- Filtra grupos não desejados;
> `ORDER BY`- Ordena as linhas da tabela não persistente de acordo com uma ou mais colunas.
